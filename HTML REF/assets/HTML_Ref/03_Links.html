Links

A normal link:
<a href="example.com">example</a>

You could also put in a computer address if you are using a html document to mange items and files on your computer

<a href="C:/music/song.mp3">example song</a>

You can do anchors that you name and then have a link to that anchor, this makes it great if you have to jump around the page, or if there is a lot to read and the reader wants to jump back to the top.

<a name="top">Top of Page</a>
Here is the name of link "top" and it is places near the top of the html document.

<a href="#top">Go back to the top!</a>
And here is the link that you take you to where the "top" link is located.

You can make an entire picture a link if you would like:
<a href="#top">
<img src="example.gif" alt="If you hover over image you get this text.l"  />
</a>
