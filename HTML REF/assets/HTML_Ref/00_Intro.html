Intro to HTML

HTML stands for Hyper Text Markup Lanuage, and is not a programming lanuage but more of a layout lanuage that describes to a browser how the document or web page should be formatted.

HTML can be edited by anything that can change text, although I would not recommend a full blow word processor since they have markeup lanuage in their documents and they might be copied over when you get ready to make your web page.  Some good editors are Notepad,  Kompozer, and Geany (for Linux).

Once you are ready to save your HTML make sure that you put the .html extention on the end of your file name.  For some editors like Kompozer it will do this for you.
